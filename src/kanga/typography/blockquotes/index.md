Use a blockquote to pull a passage out of the flow of the text. They're styled as centred pull quotes with typographic rules, set in Tiempos Fine italic.

## Attributed blockquote

Use the `<quote>` shortcode when the source should be credited. The attribution appears below the quote in a smaller monospace style. Both the content and the attribution support inline Markdown (bold, italic, links).
